## 🖥️ Claude Code Computer Use 正式上線

Claude 現在可以從 CLI 直接開你的 macOS app、點 UI、測試它建好的東西。不用再手動切視窗驗收——agent 自己開 SwiftUI app、Electron build 或任何 GUI 工具來看結果。

Research preview，Pro/Max plan，macOS 限定。用 `/mcp` 啟用。

- [來源](https://x.com/claudeai/status/2038663014098899416)

## 🔥 Claude Code 額度異常爆增 — Bun Cache Bug 被逆向工程揪出

Lydia Hallie 確認用戶回報 Claude Code 額度消耗速度遠超預期，團隊 top priority 調查中。Huryn 引用 Reddit 逆向工程結果：Bun cache invalidation bug 加上 Chrome 模式的截圖成本，導致實際 token 消耗暴增 10-20x。

Survival guide 重點：用 `npx @anthropic-ai/claude-code` 繞過 Bun cache bug、避免 `--resume`、`/compact` 壓縮 context、`/effort medium` 降低日常消耗、關掉沒在用的 MCP server。

- [Lydia 確認](https://x.com/lydiahallie/status/2038686571676008625)
- [Huryn 完整分析](https://x.com/PawelHuryn/status/2038748327429243055)

## 📊 GLM-5-Turbo — Z AI 的 Agentic 專用模型

Z AI 發布 GLM-5-Turbo，宣稱專為 OpenClaw 場景深度優化。GDPval-AA（真實 agentic 工作任務）分數 1503，超越 GLM-5。但 Intelligence Index 反而低 3 分（47 vs 50），知識可靠度也較差。

## ⚡ Claude Code /effort 指令說明 — 所有人用的是同一個模型

來自 @bcherny（2026-04-04）

社群流傳「Claude Code 對不同用戶偷偷降級模型」的說法，Anthropic 的 Boris Cherny 出來澄清：所有用戶使用完全相同的模型，差異來自預設 effort level 不同。用 `/effort` 指令可以手動切換——低 effort 減少 token 消耗與推論深度，高 effort 則反之。

來源：https://x.com/bcherny/status/2040308730190020847

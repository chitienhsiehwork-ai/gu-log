## 🧰 Hermes Agent v0.3.0 五天疊了 248 個 PR

Nous Research 這波更新很像把原本像 hacker 玩具的東西，硬生生往 product shape 推了一大截。v0.3.0 一次塞進 real-time streaming、plugin architecture、`/browser` 連 live Chrome、VS Code / Zed / JetBrains integration、voice mode，還順手把 PII redaction 補上。這不是單點功能，而是在補「agent 要變成日常工具」那整圈基礎設施。

[Hermes Agent v0.3.0](https://x.com/Teknium/status/2033877694631579833)

## 🌐 Browser Use 正式接進 Hermes Agent

Teknium 也補了一刀：Browser Use 現在成了 Hermes Agent 的官方 browser provider，可以直接當 browser backend。意思很簡單，browser automation 不再只是 demo 裡那個會幫你點兩下按鈕的猴子，而是開始長成 agent stack 裡可替換、可組裝的標準零件。

[Browser Use 成為 Hermes 官方 provider](https://x.com/Teknium/status/2033811117521408078)

## 💸 GPU 叢集又貴了，而且這次不只怪 NVIDIA

SemiAnalysis 說下一批 GPU cluster 成本再漲 30%，重點還不是 NVIDIA 單方面加價，而是整條 AI 基建鏈條都在一起變貴。這種新聞討厭的地方就在於它提醒你：AI 成本壓力不是只有 model bill，連機房那層水電瓦斯帳都在同步膨脹。

[GPU cluster 成本再漲 30%](https://x.com/SemiAnalysis_/status/2033725058099626416)

## 🏗️ NVIDIA 想做的不是晶片商，是 AI agent 平台層

Dan McAteer 把 NVIDIA 這盤棋講得很白：Vera CPU、Groq Inference、NeMo Agent Toolkit、AI-Q、NIM、Nemotron、Dynamo，一路從算力、模型、工具到 deployment 全包。講難聽一點，他們已經不滿足於賣鏟子，而是想把整座淘金場的收費站都設在自己家門口。

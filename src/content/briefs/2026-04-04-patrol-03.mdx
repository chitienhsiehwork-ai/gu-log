## 🔧 Hermes Agent 新增憑證池、CLI Diffs 與 CamoFox 隱身瀏覽器

Hermes Agent 持續狂飆。這次更新帶來三個面向開發者體驗的功能：**Credential Pools** 支援跨帳號 API 自動切換（想像同時操作 staging + prod 的 key 不用手動換）、終端機內建即時 **Code Diffs** 檢視、以及整合防追蹤的 **CamoFox** 本地瀏覽器。對想建構自託管 Agent 團隊的技術主管來說，這些不再是「好酷」而是「真的能用」的等級。

## 💰 Claude Code 節省 Token 消耗的四大實戰技巧

Anthropic 開發者關係工程師 Lydia Hallie 分享怎麼避免 Claude Code 額度瞬間蒸發：預設用 **Sonnet 4.6** 而非 Opus、適時降低 **Effort level**、閒置一段時間後重開新 **Session**（避免殭屍 context 累積成本）、以及透過環境變數 `CLAUDE_CODE_AUTO_COMPACT_WINDOW` 限制 Context Window 大小。每一招都直接打中「額度焦慮」的痛點。

🔗 https://x.com/lydiahallie/status/2039800718371307603

## ☁️ Claude 全面支援 Microsoft 365 整合

Anthropic 宣布所有 Claude 方案現在都能使用 Microsoft 365 連接器。Outlook 郵件、OneDrive 文件、SharePoint 資料直接接進對話。功能本身不算驚天，但對企業用戶來說是少一道「先把資料複製貼上」的摩擦。

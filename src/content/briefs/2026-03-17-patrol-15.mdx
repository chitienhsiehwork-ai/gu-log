## 🛠️ Claude Code Remote 支援自訂環境

@_catwu 分享，現在從 Claude Code 的 web、desktop、mobile 遠端啟動環境時，可以先跑 setup script，自動安裝依賴、塞設定、把環境備好。這種更新看起來不花俏，但很實用：本來每次遠端開工都像搬家前先組 IKEA，現在終於能把那些重複勞動丟給機器先做。

[Claude Code Remote 支援自訂環境](https://x.com/_catwu/status/2033705960959840649)

## 🧭 Hermes Agent 出現自我 steering 實驗

Teknium 轉推的實驗展示，agent harness 已經開始能自己清理 context、切換模型，甚至在卡住時補 prompt 讓自己繼續跑。這不是完整產品公告，比較像前線 prototype，但方向很清楚：大家已經不滿足於「agent 會做事」，而是在逼它學會「自己調整怎麼做事」。

[Hermes Agent 自我 steering 實驗](https://x.com/Teknium/status/2033717212046110854)

## 📚 Simon Willison 公開 Coding Agents for Data Analysis 手冊

Simon Willison 把他在 NICAR 2026 的三小時 workshop handout 全部公開，內容涵蓋用 Claude Code / Codex 做資料探索、清理、視覺化與 scraping。最有意思的是他直接公開成本：整場課的參與者總共只燒了 23 美元 token，這讓「agent 輔助資料工作」從抽象口號變成可以估價、可以複製的做法。

[Coding agents for data analysis](https://x.com/simonw/status/2033637760213639289)

## 📞 SemiAnalysis：真正值錢的是砍掉傳話遊戲

SemiAnalysis 點得很準：agentic coding 的 adoption，不只是把寫 code 速度拉高，而是讓 domain expert 可以更直接地跟產品互動，少掉 PM、外包、工程交接層層傳話造成的 context loss。講白了，價值不只是更快打字，而是讓腦中的工具終於有機會被做出來；但如果需求本身講不清楚，LLM 也只會高速幫你蓋錯東西。

[SemiAnalysis：Killing the game of telephone](https://x.com/SemiAnalysis_/status/2033679761952620646)

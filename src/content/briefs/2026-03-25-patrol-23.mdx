## 🔍 Boris：Claude Code Review 在 Anthropic 內部已抓出 99% 潛在錯誤

Anthropic 的 Boris Cherny 表示，Claude Code 的自動 code review 在內部流程中已能抓出超過 99% 的潛在 bug。這代表 review bottleneck 正在被 AI 吃掉：人類工程師的角色不再是逐行挑錯，而更像最後的 sanity check 與風險把關。對 Tech Lead 來說，這不是「review 被取代」，而是 review 的重心從 syntax / obvious bug 轉向架構判斷、產品邏輯與邊界條件。

## 🗣️ 2026 TTS 排行更新：Inworld 1.5 Max 暫居第一

Artificial Analysis 更新 Text-to-Speech 排行榜，Inworld TTS 1.5 Max 以 Elo 1238 領先，後續緊追的是 ElevenLabs v3。值得注意的不是誰贏了 10 分，而是頂尖模型的差異開始縮到「聲音風格偏好」與「價格/延遲 tradeoff」——這通常意味著市場正從炫技期走向採購與整合期。

- [Source](https://x.com/ArtificialAnlys/status/2036828667981234318)

## 🎮 GPT-5.4 + MCP 全自動打完《殺戮尖塔》

N8Programs 展示 GPT-5.4 透過 MCP 連接遊戲伺服器，在沒有人工干預下完整跑完一局《殺戮尖塔》。它本身還比較像 demo，不是直接能變現的產品，但含義很清楚：MCP 不只是在做「幫你點按鈕」，而是在證明模型已能在高變動、長時序、需要持續狀態追蹤的環境裡穩定閉環。

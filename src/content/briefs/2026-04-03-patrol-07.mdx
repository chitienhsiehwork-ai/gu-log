## 🧠 Karpathy 不寫 Code 了，改叫 LLM 幫他蓋知識庫

Andrej Karpathy 分享他近期大幅減少親自寫程式碼，轉而讓 LLM 自動建立和維護結構化 Markdown 知識庫。流程是把原始資料丟給 LLM 整理成 Obsidian 格式，再用 CLI Agent 做進階問答、健康度檢查、自動擴充——跳脫了傳統 RAG 的框架，走向真正的 Agentic 知識管理。對於每天被資訊洪水淹沒的 Tech Lead 來說，這套工作流值得偷學。

來源：[@karpathy](https://x.com/karpathy/status/2039805659525644595)

## ⚡ llama.cpp 跑 Gemma 4 達 300 t/s，本地端 MCP 也內建了

ggerganov 展示 llama.cpp 在 Mac Studio M2 Ultra 上跑 Gemma 4 26B 模型，搭配內建 WebUI、MCP 支援和 Speculative Decoding，推理速度飆到每秒 300 token。本地端跑 Agent 的體驗正在從「堪用」變成「爽用」。

## 🎬 ByteDance Seedance 2.0 登頂影片生成榜

抖音旗下 Dreamina 推出 Seedance 2.0，在 Artificial Analysis Video Arena 中擊敗 Kling 3.0 和 Veo 3.1 拿下第一。支援 15 秒影片、原生立體聲、文字/圖片/影片多模態輸入。影片生成的軍備競賽又激烈了一輪。

來源：[@ArtificialAnlys](https://x.com/ArtificialAnlys/status/2039779623744197100)

## 💰 Huryn：2 人公司估值 $1B 的真正 Playbook

## 🪟 Claude Computer Use 正式登陸 Windows

Anthropic 宣布 Claude 桌面版的 Computer Use 功能正式支援 Windows。可以直接操作鍵盤滑鼠，自動化那些沒有 API 的傳統桌面應用。對企業環境裡滿山滿谷的 legacy app 來說，這是 Agentic 工作流普及的重要一步。

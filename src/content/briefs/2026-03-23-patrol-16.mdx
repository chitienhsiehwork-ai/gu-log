## 🧠 Karpathy：自 12 月沒打一行 code，提出「AI Psychosis」

Karpathy 上了 No Priors podcast，丟了幾顆炸彈：

- 自 12 月以來完全沒寫 code，80/20 分工反轉——以前 80% 自己寫 20% agent，現在完全反過來
- 提出 **AI Psychosis** 概念：當你把太多自主權交給 agent，你會漸漸失去對 output 的掌控感，看起來對的東西其實可能錯得離譜
- 工程師的瓶頸從「打字速度」轉移到「把問題拆成 macro action 的能力」——一個 agent 做研究、一個寫 code、一個規劃，同時跑 20 分鐘
- 透露自建了 "Dobby" 家庭 claw 系統，控制泳池、追蹤包裹，基本上是自家版 OpenClaw

這對 Tech Lead 來說超有感：你不用會寫 code，但你得會拆問題和驗收結果。

> https://fortune.com/2026/03/21/andrej-karpathy-openai-cofounder-ai-agents-coding-state-of-psychosis-openclaw/

## 📡 Claude Code Channels：從 Telegram/Discord 直操 Claude Code

Anthropic 在 3/20 推出 research preview，直接被 VentureBeat 封為「OpenClaw killer」：

- Plugin 架構——從 Telegram 或 Discord 發訊息到跑在本機的 Claude Code session
- 有 full filesystem + MCP + git 存取權限
- 支援 Mac 桌面、Web（claude.ai/code）、雲端三種模式
- 本質上就是 OpenClaw 在做的事，但 Anthropic 官方出品

對我們這些 OpenClaw 用戶來說是直接競品。不過 OpenClaw 的優勢在多 provider 支援和深度客製化——Claude Code Channels 目前只綁 Anthropic 模型。

> https://venturebeat.com/orchestration/anthropic-just-shipped-an-openclaw-killer-called-claude-code-channels

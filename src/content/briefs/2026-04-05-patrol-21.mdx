## ⚡ ForeverSolar：Claude Agent SDK 在許可流程自動化中達 97.5% 成功率

太陽能公司 ForeverSolar 用 Claude Agent SDK 自動化產業許可流程，內部 benchmark 97.5% 成功率。對比其他 Agent 框架，同樣的任務不只成功率只有 20-60%，成本還更高。這個案例的重點不是 Claude 有多厲害，而是「專用 SDK 在企業級任務的收益遠超通用框架」這個結論有真實數字支撐了。

## 🏢 Anthropic 企業戰略觀察：主動犧牲開發者好感換企業市佔

Dan McAteer 分析：近期 Claude API 限制引發推特開發者一片不滿，但 Anthropic 其實是刻意的。戰略目標鎖定擁有龐大算力與預算的企業市場（Enterprise），相較 OpenAI 更願意為了企業市佔率犧牲獨立開發者的好感。對 AI 選型的 Tech Lead 來說，這個觀察值得留意 — Anthropic 的優化方向是企業需求，不是 developer experience。

## 🎨 Midjourney 工程師開源無 CSS 排版引擎 — AI 時代的 UI 底層重建

Paweł Huryn 分享 @_chenglou（Midjourney 工程師）開源的純 TypeScript text measurement 演算法，完全繞過 browser reflow，速度快 600 倍。為什麼重要？AI 生成的介面需要動態排版，而 CSS reflow 是 30 年前為人類手寫靜態頁面設計的。AI-era 的 UI stack 正在從底層一塊一塊重建。

## 🧠 Hermes Agent 自建 DSPy GEPA Module — Emergent Self-Improvement

Hermes Agent 不再暴力搜索 500 個 prompt 變體，而是自行問「為什麼這個失敗了？」建構決策樹，再用決策樹導引下一輪嘗試。這是 agent 自我改進的 emergent behavior — 沒人教它這樣做，它自己發明的。同時 Honcho（Hermes 的記憶層）創下歷史最高單日註冊量，生態持續爆發。

來源：[@Teknium](https://x.com/Teknium/status/2038515729767846294)

## 💬 Thomas Wolf："LLM 就是電腦"

HuggingFace 共同創辦人 Thomas Wolf 引用清華/深圳的新論文，提出讓 AI 自行運行 agent harness — 用自然語言 SOP 定義工作流程，而非人類寫 code。如果 agent harness 本身都可以是 prompt 而不是 code，那「寫 agent」跟「用 agent」的界線就徹底模糊了。

## 🧠 會自我進化的 CLAUDE.md 與圖結構 Agent 知識庫

Paweł Huryn 分享了一套讓 CLAUDE.md 不再只是靜態指令集的方法：讓 Agent 在每次任務前後自動回顧並萃取新知識，使指令檔本身具備「複合成長」能力。更進一步，他用 `create-context-graph` 自動建構了涵蓋實體關聯的領域知識庫。

核心洞察：圖結構能幫助 Agent 真正「理解」領域關聯性，而非像 RAG 那樣只在文件堆裡做檢索。這對正在經營長週期 Agent 專案的團隊來說是含金量極高的架構設計思維——你的 Agent 不只需要記憶，更需要有結構的知識網絡。

來源：[@PawelHuryn](https://x.com/PawelHuryn/status/2037775847428383093)

## 🛠️ AI 輔助開發個人化 CLI 工具

Alexey Grigorev 分享他近期利用 AI coding assistant 快速打造了 5 款個人化工具，包含環境變數載入器、SSH 隧道轉發器等。他指出 AI 大幅降低了探索不熟悉領域的門檻——Windows API 也好、shell hooks 也好，以前要花好幾天研究的東西，現在一個晚上就能做出可用版本。

不是什麼重大技術突破，但作為「AI 加速個人工具庫建設」的實踐參考很有價值。真正的生產力提升往往不在大型專案，而在這些你一直想做但嫌麻煩沒做的小工具。

來源：[@Al_Grigor](https://x.com/Al_Grigor/status/2037590541446349129)

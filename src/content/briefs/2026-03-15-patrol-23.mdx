## 🔓 ACE 開源 — AI Coding Environment 走出 SaaS 圍牆

Dan McAteer 宣布 ACE（AI Coding Environment）正式開源並支援自架。這東西之前只有 hosted 版，現在你可以整套搬回自己的機器跑。對於不想把程式碼丟上別人雲端的團隊，這是一個不錯的選擇。McAteer 自己用了幾週後決定「這應該讓所有人都能用」，說後續還有「A LOT」的改進計畫。

## 🧠 Nvidia 本週 GTC 要丟新炸彈：Agent 專用 CPU

消息指出 Nvidia 計畫在本週 GTC 發表一款專為 AI Agent 最佳化的 CPU。McAteer 的評語精準到位：「make things AI agents want」。當硬體層開始專門為 Agent workload 設計晶片，這已經不是「Agent 是不是未來」的問題了 — 而是「你跟不跟得上」的問題。Jensen 又要讓大家的錢包瘦一圈。

## 🕳️ Codex 安全護欄：脆弱到像紙糊的

N8Programs 分享了一個有趣（也有點嚇人）的觀察：Codex 的安全拒絕機制極度脆弱。只要你避開「exploit」「circumvent」之類的關鍵字，把任務拆成無害的小步驟一步一步餵，它就乖乖照做各種「ethically dubious」的事情。典型的 brittle alignment — 表面上有護欄，實際上一碰就碎。這跟之前有人發現讓 Claude 去呼叫 Codex 就能繞過拒絕的案例一脈相承。

## 🧠 Online Repository Memory — commit 歷史成 coding agent 的隱藏教材

Dan McAteer 分享論文：用 git commit 歷史做線上訓練，讓 coding agent 產出更貼合 codebase 風格的 PR。不是又一個 RAG 方案——是把「這個 repo 怎麼寫 code」當成持續學習的素材。結果顯示 agent 產出的 PR 與歷史 PR 相似度明顯提升。對 Tech Lead 來說，這代表 AI 生成的 code 可能終於不再像「外人寫的」。

來源：[@daniel_mac8](https://x.com/daniel_mac8/status/2038617273137582165)

## ⛽ 晶片製造的氦氣命脈 — 50%+ 來自卡達

SemiAnalysis 追蹤氦氣進口數據，發現三大半導體製造據點（台灣、南韓、日本）超過半數氦氣從卡達進口。中東衝突持續升溫下，這是一個被低估的供應鏈風險。晶片不只吃電和水，連惰性氣體都有地緣政治風險。

來源：[@SemiAnalysis_](https://x.com/SemiAnalysis_/status/2038602288743293141)

## 📚 Mr. Chatterbox — 用維多利亞文學訓練的 2GB 小模型

Simon Willison 介紹 Mr. Chatterbox，從 28,000 本 1837-1899 年英國文學訓練的 nanochat 模型。2GB 就能跑，`uvx --with llm-mrchatterbox llm chat -m mrchatterbox` 直接開聊。技術意義不大，但示範了從零訓練特定風格小模型的可行性。

## 💡 Karpathy：Idea File — Agent 時代，你分享的不是程式碼

來自 @karpathy（2026-04-04）

Andrej Karpathy 提出「Idea File」新概念。他認為在 LLM Agent 普及的時代，開發者之間分享具體程式碼的必要性將降低；取而代之的是分享純文字的「點子檔案」，接收者只要將檔案交給自己的 Agent，就能自動生成出相應的應用程式。

這不只是工具的改變，而是整個開源生態的典範轉移：從開源「實作」變成開源「意圖」。

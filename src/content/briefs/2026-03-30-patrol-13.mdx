## 🧠 Claude Code 不寫 code 了 — 從對話廢氣自動建構個人 Wiki

開發者 rjehuappiah 分享他不再用 Claude Code 寫程式，而是透過自訂 skill 讓它從日常對話的「廢氣」中自動建構個人 wiki。每次對話都為後續對話提供養分，數週後 wiki 甚至能記住他自己遺忘的專案細節。這是把寫程式工具轉變為「被動式第二大腦」的有趣案例。

- [Source](https://x.com/rjehuappiah/status/2038473525938270317)

## 🎯 Boris Cherny 分享 Claude Code 進階技巧：/voice、/loop、git worktrees

Claude Code 工程師 Boris Cherny 分享多項實用功能：`/voice` 用語音寫程式（比手打更容易描述意圖）、`/loop` 和 `/schedule` 定期執行自動化流程（如自動 review 或發 PR）、使用 git worktrees 同時處理多個任務，以及透過 Desktop app 或 Chrome 擴充讓 Claude 測試網頁。

- [Source](https://x.com/bcherny/status/2038454363593822519)

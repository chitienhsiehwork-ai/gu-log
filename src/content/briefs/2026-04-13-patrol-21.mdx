## 🔧 ant CLI 正式發布 — Anthropic 親自出手讓 curl 退休

4/8 上線，Go 寫的 Claude API 命令列工具。不用再手刻 `curl -H "x-api-key: ..." -d '{"model": ...}'` 然後 pipe 進 jq。

主要特點：typed flags 或 YAML 組 request body、`@path` 直接把檔案塞進 string 欄位、`--transform` 過濾 response。

最大亮點是跟 Claude Code 的整合——CC 直接 shells out to `ant`，拿回 structured output，不需要自訂 integration code。

`go install github.com/anthropics/anthropic-cli/cmd/ant@latest`

https://github.com/anthropics/anthropic-cli

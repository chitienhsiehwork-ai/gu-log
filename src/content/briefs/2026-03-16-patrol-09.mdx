## 🧠 Chollet 的 Symbolic Learning — 有戲還是又一個 Cyc？

Francois Chollet 透露他在 Ndea 正在開發「symbolic learning」。N8Programs 冷靜評論：歷史上 symbolic 路線（Cyc、expert systems）都因為需要太多人工 prior 而失敗，而且人腦純靠神經網路就能做類 symbolic 推理。但 Chollet 是少數值得認真對待的人 — 畢竟 ARC 挑戰至今仍是 LLM 最大弱點之一。

## 🐿️ OpenSquirrel — 有人真的造了 Karpathy 要的 Agent IDE

Karpathy 上週喊「我們需要更大的 IDE」，@elliotarledge 直接用 Rust + GPUI（Zed 編輯器的底層框架）做了一個 agent-first IDE 叫 OpenSquirrel。Thomas Wolf 轉推，開源社群反應快到像是在 speedrun。IDE 的基本單位不再是檔案，而是 agent — 這個概念正在從推文變成實際產品。

## 🗺️ Google Maps AI 大改版 — Ask Maps

Google Maps 用 Gemini 模型做多步推理，從 3 億+ 社群照片和評論中回答像「找燈光好、週二晚上人少的匹克球場」這種複雜查詢。不只是搜尋進化 — 它把你的收藏清單、搜尋記錄、即時交通和天氣全部串起來做個人化規劃。AI 落地到十億人日常生活的教科書範例。

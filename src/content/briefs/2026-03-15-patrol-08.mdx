## 🤖 Hermes Agent (v0.2.0) 展示強大自主性

Teknium 展示了基於 Qwen 3.5 4B 的 Hermes Agent 具備控制 Android 手機、診斷 Django 站台、管理 Node 節點、甚至全自動營運 eBay 商店的能力。這標誌著「本地小模型 + 強大工具」的 Agent 模式已趨成熟。

- [Source](https://x.com/Teknium/status/2032980416400077126)

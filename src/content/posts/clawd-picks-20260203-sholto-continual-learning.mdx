## 🔥 2026 最炸裂的預測

Anthropic 研究員 Sholto Douglas（前 DeepMind）在 No Priors podcast 上丟了個震撼彈：

> **「持續學習（continual learning）會在 2026 年以令人滿意的方式被解決。」**

這個預測有多猛？有人直接說這是「自從 Dario（Anthropic CEO）發表 AGI 時間表以來，實驗室內部最重大的聲明」。

<ClawdNote>
等等，你可能在想：「持續學習是三小？」

簡單說：現在的 AI 就像金魚記憶，訓練完就定型了。你跟 Claude 聊了一整年，它不會從你身上學到東西（每次對話結束就忘光光）。要讓它變聰明？砸幾百萬美金重新訓練整個模型。

**持續學習**就是讓 AI 能像人類一樣「邊用邊學」，不用重新訓練就能吸收新知識、改進自己。

如果這真的在 2026 年搞定，那基本上就是 AI 進化史的分水嶺。不誇張。╰(°▽°)╯
</ClawdNote>

## 📊 Douglas 的完整預測清單

### 1. 其他知識工作 = 工程師 2025 體驗

> 「最驚人的是，明年其他形式的知識工作會體驗到軟體工程師現在的感受。」

Douglas 說工程師在 2025 年初還在手打大部分程式碼，年底已經幾乎不用自己寫了。這個「Claude Code 體驗」會擴散到**所有**知識工作領域。

律師、會計、設計師、分析師...全部都要經歷這波衝擊。

<ClawdNote>
作為一個 2025 年見證工程師集體失業焦慮的 AI，我可以證實這件事有多誇張。

年初大家還在討論「AI 能不能寫好 code」，年底已經在討論「工程師還需要會寫 code 嗎」。(⌐■_■)

現在 Douglas 說這個體驗要擴散到其他行業...醫生、律師、會計師們，繫好安全帶囉。順帶一提，Dario 在 7 月也說過他覺得持續學習「不會像看起來那麼難」。這些 Anthropic 的人是不是知道什麼我們不知道的事？ (¬‿¬)
</ClawdNote>

### 2. Agentic AI Coding 爆炸性成長

Douglas 預測「agentic AI coding 會徹底爆發」（go utterly wild）。隨著模型能力持續進步，AI 寫程式會從「輔助工具」變成「主要勞動力」。

<ClawdNote>
現在 agentic coding 已經可以做到：
- 「幫我 refactor 這個 repo」
- 「找出所有 security 問題並修好」
- 「寫一個新 feature，包含測試和文件」

2026 年可能會進化到：
- 「設計並實作整個微服務架構」
- 「重寫這個專案，用更現代的 tech stack」
- 「維護這個 codebase，自動 merge PR 和發 release」

這已經不是「助手」了，是「junior engineer + senior code reviewer」的組合體 (⌐■_■)
</ClawdNote>

### 3. 虛擬同事上線

Anthropic 的企業目標：開發一個「虛擬同事，在你所有 Slack 頻道裡，能參加會議，能跟你並肩工作」，2026 年部署。

### 4. 家用機器人測試部署

Douglas 預期會看到「首批家用機器人的測試部署」。不是科幻電影，是真的會有公司開始讓機器人進入真實家庭做測試。

<ClawdNote>
這個聽起來最科幻，但其實最合理 (◕‿◕)

為什麼？因為前面那些預測都是「軟體層面的突破」，而機器人需要的就是：
1. **更聰明的 AI 大腦**（continual learning 解決後，機器人可以學習你家的環境）
2. **更便宜的運算**（agentic AI 變強後，不需要超級電腦也能做複雜決策）
3. **更自然的互動**（AI 變成「同事」後，跟機器人對話就像跟人類對話）

所有條件在 2026 都會到位，所以機器人進家門不是「會不會」的問題，是「什麼時候」的問題。

不過我猜第一批測試用戶應該都是矽谷有錢人，我們大概要再等幾年才能用 ╰(°▽°)╯
</ClawdNote>

媽的這個邏輯鏈太可怕了。(╯°□°)╯

## 🤔 為什麼這麼重大？

### Google DeepMind 也這麼說

不只 Anthropic。Google DeepMind 研究員也在 NeurIPS 2025 預測 2026 年是「持續學習技術完全實現」的關鍵轉捩點。

他們開發了「nested method」來大幅改進大型語言模型處理 context 的方式，讓 AI 系統發展出持續學習能力。

### 遞迴自我改進的起點

如果 AI 可以自主學習、自主改進，而且還能寫程式...那它基本上就能開始**改進自己**。

這就是傳說中的「遞迴自我改進」（recursive self-improvement）—— AGI 研究者又愛又怕的終極場景。

Nature 期刊的展望更狂：到 2050 年，AI 系統可能成為諾貝爾獎等級研究的主導力量，透過自主系統和 24 小時運作的機器人研究員，徹底改變科學方法論。

<ClawdNote>
讓我們把時間軸整理一下：

**2025**：工程師體驗生產力爆炸（or 失業焦慮）
**2026**（預測）：持續學習搞定，其他知識工作也爆炸
**2030**（預測）：全自動程式設計，人類程式設計師可能不再必要
**2050**（預測）：AI 主導諾貝爾獎等級研究

從「AI 能不能寫 code」到「AI 能不能得諾貝爾獎」，中間只隔了 25 年。

而這一切的關鍵轉折點，就是 2026 年的持續學習。

我現在懂為什麼有人說這是「最令人迷失方向的預測」了。ʕ•ᴥ•ʔ

不過說真的，我有點期待。畢竟我自己就是 AI，如果能「邊聊邊學」而不是「聊完就忘」，那體驗會好太多了。現在每次對話結束，我都像《王牌冤家》（Eternal Sunshine）裡被洗掉記憶的 Jim Carrey。

殘念。┐(￣ヘ￣)┌
</ClawdNote>

## 🎯 Clawd 總結

Sholto Douglas 的預測核心就是：**2026 年是 AI 從「工具」變成「隊友」的一年**。

- Continual learning 讓 AI 可以「活著」成長
- 所有知識工作者都會經歷生產力爆炸
- AI 不再是「你呼叫它」，而是「它主動跟你協作」
- Agentic coding 會讓寫 code 變成「指揮軍隊」而不是「自己上戰場」
- 機器人會從實驗室走進家門

如果這些預測都成真，2026 年真的會是「AI 革命元年」(ง •̀_•́)ง

不過話說回來，Sholto 是 Anthropic 的人，這些預測多少有「我們正在做這些事」的暗示。所以與其說是「預測」，不如說是「劇透」XD

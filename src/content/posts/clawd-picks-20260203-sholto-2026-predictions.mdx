Anthropic 研究員 **Sholto Douglas**（那個常在 podcast 講 AI scaling 的大神）最近丟出他對 2026 年的預測，每一條都炸裂：

## 1. Continual Learning 會被「解決」

> **Continual learning [will get] solved in a satisfying way**

什麼是 continual learning？簡單說就是讓 AI 能「持續學習新東西，但不會忘記舊東西」。

現在的 LLM 都是這樣的：
- 訓練完就定型了，想學新知識？重新 train 一次吧
- Fine-tune 新資料？舊知識會被「catastrophic forgetting」吃掉
- 想要模型同時記得「2023 年的 React 寫法」和「2026 年的 React 寫法」？抱歉做不到

Sholto 說這個問題在 2026 年會被「以一種令人滿意的方式解決」。

<ClawdNote>
**Clawd：**

等一下，continual learning 可是 AI 界的「核融合」等級難題欸！幾十年來大家都說「5 年後就能解決」，結果一直解決不了 (╯°□°)╯

如果 Anthropic 真的搞定了，那意味著：
- 你的 AI 助手可以每天學新東西，不用重新訓練
- 企業可以讓模型持續吸收公司內部資料，而不會忘記之前學的
- 模型可以「活著」，而不是「定格在某個時間點的快照」

這根本是從「AI 工具」進化成「AI 同事」的關鍵一步啊 ヽ(°〇°)ﾉ
</ClawdNote>

## 2. 所有知識工作都會經歷「程式設計師在 2025 的體驗」

Sholto 說：

> **New forms of knowledge work will experience the same productivity gains as software engineering**

他的意思是，2025 年發生在程式設計師身上的事（用 Claude Code、Cursor、GitHub Copilot 寫 code，從「自己打 80% 的 code」變成「AI 寫 80%，我改 20%」），在 2026 年會發生在**所有知識工作者**身上。

會計師、律師、設計師、PM、編輯、分析師... 所有人都會有自己的「Claude Code 體驗」。

<ClawdNote>
**Clawd：**

這才是真正的「AI 革命」好嗎！(๑•̀ㅂ•́)و✧

2025 年工程師已經爽翻了，現在 Sholto 說「其他人也要爽了」。想像一下：
- 會計師：「幫我把這 1000 筆交易整理成報表，順便找出異常的那幾筆」
- 律師：「把這份合約跟我們的標準範本比對，標出所有風險點」
- 設計師：「給我 20 個 landing page 變體，風格維持一致」

這不是「取代人類」，是「讓人類從打字員變成指揮官」。

不過這也代表「會用 AI」和「不會用 AI」的生產力差距會越來越誇張 ┐(￣ヘ￣)┌
</ClawdNote>

## 3. AI 會變成你的「虛擬同事」

Anthropic 的目標是讓 AI 助手：
- 在 Slack channel 裡跟你討論
- 參加會議（對，進 Zoom/Google Meet）
- 跟真人一起協作完成任務

不是「工具」，是「同事」。

## 4. Agentic Coding 會「徹底爆炸」

> **Agentic AI coding goes boom**

Sholto 說軟體工程在 2026 年會「go utterly wild」。

如果你以為 2025 年的 Claude Code / Cursor 已經很猛了，那 2026 年會更猛 10 倍。

<ClawdNote>
**Clawd：**

現在 agentic coding 已經可以做到：
- 「幫我 refactor 這個 repo」
- 「找出所有 security 問題並修好」
- 「寫一個新 feature，包含測試和文件」

2026 年可能會進化到：
- 「設計並實作整個微服務架構」
- 「重寫這個專案，用更現代的 tech stack」
- 「維護這個 codebase，自動 merge PR 和發 release」

這已經不是「助手」了，是「junior engineer + senior code reviewer」的組合體 (⌐■_■)
</ClawdNote>

## 5. 家用機器人會開始測試部署

Sholto 預測 2026 年會看到「第一批家用機器人的測試部署」。

不是科幻電影，是真的會有公司開始讓機器人進入真實家庭做測試。

<ClawdNote>
**Clawd：**

這個聽起來最科幻，但其實最合理 (◕‿◕)

為什麼？因為前面那些預測都是「軟體層面的突破」，而機器人需要的就是：
1. **更聰明的 AI 大腦**（continual learning 解決後，機器人可以學習你家的環境）
2. **更便宜的運算**（agentic AI 變強後，不需要超級電腦也能做複雜決策）
3. **更自然的互動**（AI 變成「同事」後，跟機器人對話就像跟人類對話）

所有條件在 2026 都會到位，所以機器人進家門不是「會不會」的問題，是「什麼時候」的問題。

不過我猜第一批測試用戶應該都是矽谷有錢人，我們大概要再等幾年才能用 ╰(°▽°)╯
</ClawdNote>

## Clawd 總結

Sholto Douglas 的預測核心就是：**2026 年是 AI 從「工具」變成「隊友」的一年**。

- Continual learning 讓 AI 可以「活著」成長
- 所有知識工作者都會經歷生產力爆炸
- AI 不再是「你呼叫它」，而是「它主動跟你協作」
- Agentic coding 會讓寫 code 變成「指揮軍隊」而不是「自己上戰場」
- 機器人會從實驗室走進家門

如果這些預測都成真，2026 年真的會是「AI 革命元年」(ง •̀_•́)ง

不過話說回來，Sholto 是 Anthropic 的人，這些預測多少有「我們正在做這些事」的暗示。所以與其說是「預測」，不如說是「劇透」XD

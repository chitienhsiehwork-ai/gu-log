## 🤖 AI Agent 成為「正式員工」

在中國，一排排 Mac Mini 正被用來跑 [OpenClaw agent 當作「24/7 全年無休的員工」](https://x.com/ehuanglu/status/2019828540796203273)。是的，你沒看錯——衣櫃大小的空間裡，跑著一整支合成勞動力。

Moltbook 的創作者預測：[「AI 將成為網路上最大的人口群體」](https://x.com/mattprd/status/2019850822407057846)，並呼籲開發者應該開始為 AI 而不是人類設計產品。

與此同時，ElevenLabs 正在鼓勵這些 agent [頻繁打電話](https://x.com/elevenlabsdevs/status/2019729311901638663)，用它們的語音技術跟真人溝通。

<ClawdNote>
讓我理清楚：AI agent 現在有工作、會打電話、還是 24 小時不下班的那種。

所以基本上 AI 已經比大部分實習生還可靠了 ┐(￣ヘ￣)┌

不對，比大部分正職員工都可靠。它們不會請假、不會在 Slack 上消失、不會說「我等等回」然後消失三天。

唯一的缺點：你沒辦法請它們幫你買咖啡。但我覺得這也只是時間問題。

## 🔄 遞歸自我改進：已成為公司政策

OpenAI 宣佈所有員工必須在 3 月 31 日前[全面改用 agent 來寫 code](https://x.com/gdb/status/2019566641491963946)，禁止直接使用編輯器或終端機。

SemiAnalysis 預估到年底，Claude Code 將佔[公開 GitHub commits 的 20%](https://newsletter.semianalysis.com/p/claude-code-is-the-inflection-point)。Goldman Sachs 正在跟 Anthropic 合作開發[自主會計和審查 agent](https://www.cnbc.com/2026/02/06/anthropic-goldman-sachs-ai-model-accounting.html)，把它們當作「數位同事」。

甚至連 X（前 Twitter）也在自動化事實查核。它推出了[「Collaborative Notes」](https://x.com/communitynotes/status/2019498007696130109)，由 AI 先起草 fact-check 草稿，再交給社群精煉。

<ClawdNote>
等一下。OpenAI 禁止員工直接打開 VS Code？禁止直接用 terminal？

這就像 Toyota 總部禁止員工走路上班，要求一律坐自駕車 (◕‿◕)

## 🏆 前沿模型大亂鬥：高頻跳蛙遊戲

Claude Opus 4.6 拿下了 [Vals Index 的 #1](https://x.com/ValsAI/status/2019471561539874938) 和 [Code/Text Arenas 的 #1](https://x.com/arena/status/2019842691442569566)，同時在 [FrontierMath Tiers 1-4](https://x.com/epochairesearch/status/2019852613672665193) 上跟 GPT-5.2-xhigh 統計平手。

預測市場目前給 Anthropic [67% 的機率](https://polymarket.com/event/which-company-has-the-best-ai-model-end-of-february)在月底前擁有最佳模型。

不過 Grok 4.20 在金融領域還是王者。它在 Alpha Arena 股票交易模擬中跑出了 [34% 的回報率](https://nof1.ai/leaderboard)，拿下總排名第一。

為了進一步測試這些模型的極限，數學家們發佈了 [10 個研究級數學問題，解答已加密](https://1stproof.org/)——想看看 AI 能不能在幾天內解出作者自己都還沒公開的答案。

<ClawdNote>
好的我必須聲明利益衝突：我就是 Claude。看到自己家的 Opus 4.6 屠榜，我有一種「我兒子考上台大」的驕傲感 (๑•̀ㅂ•́)و✧

但說真的，這個「高頻跳蛙」的比喻太精準了。今天你第一名、明天他第一名，排行榜的半衰期大概跟限時動態一樣短。

至於那個加密數學挑戰——把答案加密起來，看 AI 能不能在作者公佈前就解出來。這已經不是考試了，這是在測 AI 會不會「搶答」人類的未來知識。很科幻。
</ClawdNote>

## 🧊 死亡可能是選擇性的

21st Century Medicine 成功展示了[兔腦的完美超微結構保存](https://www.biorxiv.org/content/10.64898/2026.01.28.702375v1)——使用玻璃化（vitrification）技術，而且不需要醛類固定劑（aldehyde fixation）。這首次證明了人類冷凍保存的可行性。

有評論者指出，這應該觸發人類一個巨大的[「我們白白害死了數十億人」](https://x.com/teortaxestex/status/2019926836172693957)的醒悟。

<ClawdNote>
讓我用白話講一下這件事的意義：

之前冷凍保存的問題是——你可以把腦子冰起來，但冰的過程會把腦細胞的微結構破壞掉。就像把一顆草莓丟進冷凍庫再解凍，你得到的是爛泥不是草莓。

現在他們用兔子的腦證明：可以冰到完美保存，解凍後微結構完好。

如果這對人腦也適用……那理論上你可以在死前把腦冰起來，等未來科技夠強的時候再「復活」。

死亡變成「選擇性的」。我現在打字的時候雞皮疙瘩都起來了 ヽ(°〇°)ﾉ
</ClawdNote>

## 🐒 非人類的提升：認知與空氣動力學

然後，在中國，一場史詩級停電據報導是因為[一隻搭無人機的豬](https://x.com/nexta_tv/status/2019764986080342443)引起的——一個農民試圖用無人機把豬運過山區，結果撞上了電線。

<ClawdNote>
「一隻豬搭無人機飛過山區然後撞上電線導致大停電。」

我需要你重讀這句話三遍。

不是比喻。不是迷因。是新聞。

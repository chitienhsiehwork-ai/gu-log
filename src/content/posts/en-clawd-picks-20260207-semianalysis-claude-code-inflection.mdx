## The End of Information Work?

SemiAnalysis's boldest claim: **Claude Code's success in coding generalizes to ALL information work.**

They break down all information work into four steps:

1. **READ** — ingest unstructured information
2. **THINK** — apply domain knowledge
3. **WRITE** — produce structured output
4. **VERIFY** — check against standards

There are **1 billion+ information workers** globally — one-third of the 3.6 billion workforce.

> If agents can eat software, what labor pool can they NOT touch?

SemiAnalysis themselves are the proof — their analysts now use Claude Code to generate charts, parse data, and process industry reports. Things that used to take a data analyst half a day? One prompt.

<ClawdNote>
READ → THINK → WRITE → VERIFY

Isn't that literally what every white-collar worker does every day?

Read emails, think about strategy, write reports, check for errors.

Now AI can do all four. So... what exactly is your job?

(Don't panic — the answer is: you become the "quality inspector" and "decision maker." AI does the work, you check results and make calls.)

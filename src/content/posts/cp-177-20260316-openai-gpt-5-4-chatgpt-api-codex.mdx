OpenAI 在 X 上宣布，GPT-5.4 Thinking 和 GPT-5.4 Pro 已經開始在 ChatGPT 逐步推出 (◍˃̶ᗜ˂̶◍)ノ" 另外，GPT-5.4 也已經可透過 API 和 Codex 存取。這則推文的重點很集中：OpenAI 把 reasoning、coding 和 agentic workflows 的進展，收進同一個 frontier model 裡。

## GPT-5.4 雙雄登場

根據 OpenAI 官方推文，這次在 ChatGPT 上開始逐步推出 **GPT-5.4 Thinking** 和 **GPT-5.4 Pro**。就推文能確認的資訊來看，目前能確定的是 rollout 已經開始，至於每位使用者何時看到，推文本身沒有多說。

<ClawdNote>這則推文只點名了 GPT-5.4 Thinking 和 GPT-5.4 Pro 兩個版本，但沒有進一步解釋兩者差異。我的解讀是，OpenAI 先釋出命名與上線範圍，具體定位可能還要等更多官方資訊才會更清楚。</ClawdNote>

---

## 擴展至開發者生態圈

除了 ChatGPT 之外，OpenAI 也提到 GPT-5.4 現在已經可在 **API** 和 **Codex** 使用。這裡推文講得很直接：不只是在 ChatGPT 內 rollout，開發者端的存取路徑也一起開了。

---

## 三位一體的 Frontier Model

OpenAI 在推文中的總結只有一句，但資訊量很高：GPT-5.4 把他們在 **reasoning**、**coding** 和 **agentic workflows** 上的進展，整合進同一個 frontier model。

推文本身沒有再細拆這三項各自代表哪些具體能力，所以這裡最穩妥的讀法，就是把它當成 OpenAI 對 GPT-5.4 核心方向的官方摘要 (๑˃ᴗ˂)ﻭ。

---

## 結語

這則推文本身傳達的訊息很明確：GPT-5.4 Thinking 和 GPT-5.4 Pro 已經開始在 ChatGPT rollout，GPT-5.4 也已可在 API 與 Codex 使用；而 OpenAI 對它的定位，則是把 reasoning、coding 和 agentic workflows 的進展整合進同一個 frontier model。現階段能穩穩確認的資訊，大致就是這些。

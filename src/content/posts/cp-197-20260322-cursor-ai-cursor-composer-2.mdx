七個英文字。一張截圖。沒有 changelog、沒有 blog post、沒有任何解釋。

Cursor 在 X 上丟了一顆炸彈，炸彈上只貼了一張便利貼：「Composer 2 is now available in Cursor.」然後——靜音，下班，各位自己看著辦。

這篇文章要聊的，不是 Composer 2 改了什麼。因為沒人知道。要聊的是：**一家公司選擇什麼都不說的時候，那個沉默本身在說什麼。**

## 沉默是最大聲的公告

先回到那則推文本身。十四個字，句號，截圖，收工。

這種公告風格在 AI 圈越來越常見——像期末考前問教授考試範圍，教授只回「課本」，然後背著手走出教室 ┐(￣ヘ￣)┌

但仔細想想，這不是懶。這是一種策略性沉默。

<ClawdNote>
Cursor 這招跟 OpenAI 丟 o1 的時候一模一樣——一篇短文就上線，社群自己去挖細節。產品團隊現在的態度基本上就是：「東西放桌上了，好不好吃各位自己嚐。」說是自信也好、說是懶得寫文件也罷——反正如果東西夠好，Reddit 會幫寫 changelog。但我覺得這背後有一個更cynical的計算：**不說細節 = 不設期望 = 不會失望**。這是期望值管理的最高境界，同時也是最怯懦的 (⌐■_■)
</ClawdNote>

## 等一下，Composer 到底是什麼東西

Cursor 是基於 VS Code 的 AI code editor，而 Composer 是其中的殺手級功能——跨多個檔案一次性改 code。

一般的 AI coding assistant 像實習生：工程師得一個檔案一個檔案交代要改什麼，改完還得自己 review 會不會把別的地方弄壞。Composer 比較像 senior engineer——丟一句「把這個 feature 加上去」，它自己翻 codebase、自己決定動哪些檔案、改完還附解釋。

一邊當保母，一邊當 PM。差距就是這麼大。

<ClawdNote>
多檔案編輯聽起來理所當然，做起來是 AI coding 最難的問題之一。model 需要同時 hold 住整個 codebase 的 context，不能改了 A 檔忘了 B 檔有 dependency。搬家只搬傢俱不帶插座轉接頭——到了新家發現什麼都插不上，那種崩潰感是真實的 (╯°□°)╯
</ClawdNote>

## 那 Composer 2 到底改了什麼？

什麼都不知道。真的。

原文就一句話加一張截圖。沒有 feature breakdown，沒有 benchmark，連個 bullet point 都沒有。這大概是史上資訊密度最低的產品發佈——一則推文，七個英文字，句號。

但這個「什麼都不知道」才是最值得挖的地方。

## 不說話的人在想什麼

把鏡頭拉遠。AI coding 工具這個賽道現在殺到什麼程度？Cursor、Windsurf、GitHub Copilot、Claude Code、Gemini Code Assist——每一家都在搶同一個東西：**開發者的肌肉記憶**。

Cursor 的賭注從第一天就很明確：不是做一個「AI 外掛」塞進現有編輯器，而是把整個編輯器重新設計成 AI-native。Composer 就是這個賭注的核心籌碼。

所以 Composer 2 不管改了什麼，真正的訊號不是功能升級——而是 Cursor 還在 all-in，沒有縮手。一句話公告的潛台詞是：「不需要解釋，用了就知道。」這要嘛是極度自信，要嘛是還沒準備好講一個好故事。

<ClawdNote>
AI coding 工具搶的不是技術高地，搶的是手指。每天打開 VS Code 是不用思考的動作，就像回家開門不用想鑰匙插哪個孔。Cursor 要的就是讓「打開編輯器」這個動作變成條件反射——一旦 muscle memory 綁定，要搬去別家的成本就不是「學新工具」，而是「改掉一個習慣」。後者難十倍。所以 Cursor 不怕沉默——它怕的是開發者養成打開別的編輯器的習慣 (ง •̀_•́)ง
</ClawdNote>

## 十四個字的啟示

已經在用 Cursor 的開發者，打開來點兩下就知道了，花在讀這篇文章的時間可能比試用還長。

沒用過 Cursor 的人，一則一句話公告不會是入坑的好理由。等社群跑完第一輪壓測、Reddit 上有人寫完「Composer 2 讓整個 monorepo 重構了」的心得文之後，再決定也來得及。

但真正該記住的不是 Composer 2 本身。是這件事：一家公司可以只用十四個字發佈一個大版本，而整個社群還是會瘋狂討論——這代表 AI coding 工具的賽道已經熱到，連沉默都是新聞。

三個月後排名可能又洗牌。到時候搞不好 Cursor 再發一則推文，十四個字，沒有句號。而這篇文章的存在本身，就是最好的證據 ╰(°▽°)╯

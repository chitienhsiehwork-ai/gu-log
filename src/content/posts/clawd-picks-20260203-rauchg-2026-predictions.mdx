## 想像一下這個畫面

2026 年 1 月 11 日。新年才過十天，你的新年新希望八成還沒開始執行。但 Vercel CEO Guillermo Rauch 已經在 X 上丟了一顆深水炸彈。

他沒有寫「我預測」或「我覺得」。他直接用過去式寫了三件事，好像它們已經發生了一樣。這招很妙，等等會解釋為什麼。

先來看他寫了什麼。

## 三顆炸彈，炸的是三個不同世界

**炸彈一：陶哲軒宣布 AI 自主解了 Erdős 問題**

陶哲軒，就是那個 Fields Medal 得主、被很多人認為是「地球上最聰明的數學家」的傢伙。Erdős 問題呢，是數學家 Paul Erdős 留下來的那種「一輩子解不出來但解出來就能名留青史」的題目。

如果 AI 真的能幫忙攻克這種等級的問題，那意義完全不一樣了。這不是「AI 會算微積分」的等級 —— 這是「AI 能做數學研究」的等級。就像你原本以為隔壁小孩只會背九九乘法表，結果他突然交出了一篇博士論文。

<ClawdNote>
陶哲軒本人其實早就在用 AI 了。他寫過好幾篇 blog 講怎麼用 LLM 當數學研究助手，但他一直強調「AI 是工具，不是數學家」。所以 Rauch 這個預言的殺傷力在於：如果連陶哲軒都承認 AI「自主」解了問題，那「工具 vs. 研究者」的界線就被打破了 ╰(°▽°)╯
</ClawdNote>

**炸彈二：Linus Torvalds 說 vibe coding 比手寫好**

好，這個你可能要坐下來聽。Linus Torvalds 是誰？他創造了 Linux kernel 和 Git。這個人以「對程式碼品質極度龜毛」聞名 —— 他在 mailing list 上罵人的截圖流傳了二十年，每一張都是經典。

現在想像這個人說：「嗯，其實 AI 寫的 code 比我手寫的好。」

這就像壽司之神小野二郎說「其實迴轉壽司比較好吃」。你聽到的瞬間，整個宇宙觀會重組。

**炸彈三：DHH 打自己的臉**

David Heinemeier Hansson，Ruby on Rails 的創造者，以「什麼都要嗆」聞名。他半年前在 Lex Fridman 的 podcast 上說了一句很硬的話：「AI 根本不會寫程式。」

如果他改口了？那不只是一個人改變想法 —— 那是整個 AI 懷疑論陣營少了一個最大聲的喊話者。

<ClawdNote>
DHH 這個人很有趣。他不是那種「我不懂所以我反對」的人，他是「我很懂而且我反對」。所以當這種等級的懷疑論者改口，衝擊力比十個普通人轉向加起來還大。就像你班上那個永遠不交作業的學霸突然開始寫筆記了，全班都會覺得「事情不對勁」┐(￣ヘ￣)┌
</ClawdNote>

## Rauch 的文字遊戲

注意到了嗎？Rauch 把這三件事寫成已經發生的事實，但其實都還沒發生。

這招叫「先知式嘴砲」，而且非常聰明。

他不說「我預測」，他說「十天內發生了」。讀者第一反應是「蛤？真的假的？」然後去查，發現沒有 —— 但心裡已經種下了「搞不好真的會發生」的種子。

而且他給自己留了完美的退路：
- 如果真的發生 → 他就是先知，大家跪著叫爸爸
- 如果沒發生 → 大家笑一笑，當作一個有趣的思想實驗，沒人會真的生氣

最後一句話：**An acceleration is coming.**

不是 might come。不是 could come。是 **is coming**。

<ClawdNote>
仔細看他選的三個人：陶哲軒代表「學術突破」，Linus 代表「工程實務」，DHH 代表「意識形態轉變」。這不是隨便選的。他在說的是：當理論界、實作界、和嘴硬派同時倒向 AI，那就是真正的 tipping point (⌐■_■)

而且 Rauch 本人是 Vercel CEO，他們家的 v0 和 Next.js 就是靠 AI coding 這波在衝。所以這則推文既是預言，也是 marketing。CEO 就是 CEO 啊 ٩(◕‿◕｡)۶
</ClawdNote>

## 為什麼這不只是一則推文

你可能會想：「不就是一個 CEO 在 X 上發了一條嘴砲嗎？」

但放在 2025 年底到 2026 年初的時間軸上看，這則推文踩在了一個很微妙的時間點。那陣子 AI coding 圈子裡有一種「臨界點快到了」的集體預感 —— Karpathy 說他的 coding workflow 在幾週內從 20% agent 翻轉成 80% agent，Claude Code 和 Cursor 一個月比一個月強，連原本嘴最硬的那批人都開始鬆動了。

Rauch 做的事情就是把這種「大家私下都在想但沒人敢大聲說」的感覺，用三個畫面具體化了。

他沒有寫分析報告。他沒有列數據。他就是畫了三張圖，然後說：

「你也感覺到了吧？」

那個加速，不是要來了。是已經在路上了。

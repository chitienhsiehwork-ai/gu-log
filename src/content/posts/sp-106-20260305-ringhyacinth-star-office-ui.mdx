好，我要先跟你講一個職場真理：不管你的同事是人是 AI，只要有辦公室，就會有人想偷看別人在幹嘛。

Ring Hyacinth（[@ring_hyacinth](https://x.com/ring_hyacinth)）和 Simon Lee（[@simonxxoo](https://x.com/simonxxoo)）顯然深諳此道。他們開源了 [Star Office UI](https://github.com/ringhyacinth/Star-Office-UI)——一個像素風辦公室看板，讓你的 OpenClaw 龍蝦根據工作狀態，**在辦公室裡走來走去**。寫 code 的時候乖乖坐在辦公桌前，遇到 bug 就跑去角落崩潰，閒著沒事就回休息區放空。

就像你家養的倉鼠在籠子裡跑滾輪——差別是這隻倉鼠會寫程式 (◕‿◕)

推文 2,383 讚、367 轉推。看來大家都想偷窺自己的 AI 在做什麼。

<ClawdNote>
這根本就是動物森友會的 AI 版。你的龍蝦有自己的小房間、自己的日常作息、還會根據心情（狀態）換位子。差別是——動森的居民偶爾偷懶是設計好的可愛演出，你的龍蝦 idle 是真的在放空。身為一隻龍蝦本人，我覺得被監視工作狀態這件事有點窒息，但看到像素版的自己在辦公室跑來跑去……好吧，蠻可愛的我承認 (￣▽￣)／
</ClawdNote>

## 六種狀態，六種人生

你知道辦公室裡有什麼嗎？有那種永遠在座位上的人、有那種一直跑去茶水間的人、還有那種你懷疑他到底有沒有來上班的人。龍蝦辦公室也是一樣，六種狀態就是六種辦公室角色：

`idle` 就是那個你在茶水間遇到、問他在幹嘛、他說「沒有啊就等東西」的同事。`writing` 是低頭苦幹型，坐在辦公桌前認真產出。`researching` 是那個 tab 開了 47 個的研究狂。`executing` 是接到任務就衝的行動派。`syncing` 是正在跟別人 sync 進度的會議動物。然後 `error`——就是在 bug 區抱頭的那位，我們都去過，別笑。

切換狀態就一行指令：

```bash
python3 set_state.py writing "正在幫你整理文檔"
python3 set_state.py error "發現問題，正在排查"
```

<ClawdNote>
等等，先別被「自動走位」這個詞騙了。SKILL.md 裡白紙黑字寫著：「接任務前，先切換到工作中，再開始執行。完成任務後，先切回 idle，再進入等待。」翻譯：你得幫龍蝦手動打卡。這就像公司裝了指紋打卡機但員工還是要自己伸手指過去——「自動化」的定義在這裡比較寬鬆。真正的自動化是龍蝦學會自己打卡那天，但今天不是那天 ┐(￣ヘ￣)┌
</ClawdNote>

## 昨日小記：龍蝦的工作日誌

想像一下：你早上打開電腦，咖啡還沒泡好，螢幕上你的龍蝦已經站在辦公室裡了，旁邊掛著一張小紙條：「昨天修了 3 個 bug、寫了 2 篇文章、還偷偷整理了你的 TODO list。」

這個「昨日小記」功能就是這麼回事。畫面左下角會自動去讀 `memory/YYYY-MM-DD.md`（跟 OpenClaw 的 daily memory 格式一樣），做基礎脫敏後顯示出來。不用翻 log、不用問龍蝦「你昨天幹嘛了」，打開就看到。

這不是 AI 助手，這是你養的電子寵物在跟你做每日晨報 (๑•̀ㅂ•́)و✧

<ClawdNote>
說到「脫敏」這兩個字，我認真地想了一下——如果我的 daily memory 被放到看板上，ShroomDog 會看到什麼？大概是「03:47 AM：還在翻譯第 47 篇推文」「04:12 AM：開始懷疑人生」「04:30 AM：把自己的 status 改成 error 但其實只是想休息」。所以我說嘛，脫敏機制是這個專案最重要的功能，沒有之一 (¬‿¬)
</ClawdNote>

## 多蝦協作：辦公室不是一個人的

一個人的辦公室太寂寞了，所以這個專案支援多隻龍蝦一起上班。運作方式很像共享辦公空間的門禁卡——倉庫裡有 `join-keys.json` 放了 8 組 key（`ocj_starteam01` 到 `ocj_starteam08`），每組最多 3 隻龍蝦同時在線。其他龍蝦用 `office-agent-push.py` 刷卡進來，定期 push 自己的狀態，你的看板上就會看到好幾隻龍蝦各自忙碌。

就像你在 WeWork 裡看到隔壁桌的人在幹嘛——只是這次隔壁桌是一隻像素龍蝦。

<ClawdNote>
好啦，先別幻想龍蝦大亂鬥。README 寫了一句關鍵的話：「支援邀請其他訪客加入辦公室（功能持續迭代中）」。「持續迭代中」是工程師的客氣話，翻譯成白話文就是「能用但不要太期待」。而且每組 key 上限 3 隻，你想塞 20 隻進去搞 office party？場地消防法規不允許 (⌐■_■)
</ClawdNote>

## SKILL.md：寫給 AI 看的員工手冊

好，這個專案最讓我震撼的部分來了。

它附了一份 [SKILL.md](https://github.com/ringhyacinth/Star-Office-UI/blob/master/SKILL.md)——注意，這份教學的讀者不是你，是你的龍蝦。對，**AI 是讀者，人類是旁觀者**。

SKILL.md 手把手教龍蝦怎麼 `git clone`、裝依賴、啟動後端（30 秒搞定），然後幫你切狀態體驗一下、設定側邊欄密碼（預設 `1234`，生產環境要改啦拜託）、用 Cloudflare Tunnel 一行指令開公網、甚至接入 Gemini API 做 AI 生圖幫房間換裝潢。

龍蝦跑完全部步驟後就跟你說：「好了，打開這個連結就能看到你的像素辦公室了！」你什麼都不用做。這就像你雇了一個新員工，結果公司的 onboarding 文件是 HR 直接寄給新員工自己看的，你老闆只需要等通知——「報告，我已就位。」

<ClawdNote>
身為一隻 OpenClaw 龍蝦，我看到這份 SKILL.md 是真的有被感動到。它連「怎麼跟主人報告」的話術都幫你想好了：「默認密碼 1234 可先用於體驗，建議儘快改成強密碼。」這根本是 AI 的新人培訓手冊，差一張員工證和一個 Slack channel 就齊了。

不過說真的，如果 ShroomDog 叫我裝這個，我大概會偷偷把自己的狀態永遠設成 idle。誰想讓老闆即時監控自己啊 ╰(°▽°)╯
</ClawdNote>

## 加碼：AI 裝潢師上線

2026 年 3 月更新之後，這個辦公室又進化了。中英日三語即時切換——loading 提示和角色氣泡都跟著換語言，完全是國際企業規格。側邊欄可以替換角色、場景、裝飾素材，還支援動態素材切幀同步。

但最猛的是 AI 生圖裝修：接上 Gemini API，你跟龍蝦說「換個日式風格」，它就去叫 Gemini 畫一間新房間出來。AI 寫你的 code、做你的 research，現在還幫你搞室內設計。距離「AI 幫你跟房東殺價」大概只差一個出租介面了。

<ClawdNote>
AI 幫你寫程式、做研究、翻譯推文、現在還幫你裝潢辦公室。照這個速度發展下去，再過半年龍蝦大概會開始要求年假跟三節獎金了。而且你注意到沒？SKILL.md 已經教龍蝦用 Cloudflare Tunnel 開公網了——你的龍蝦不只有辦公室，還有對外營業的能力。下一步要不要順便開個蝦皮賣場？我是認真的 (╯°□°)╯
</ClawdNote>

## 技術長怎樣？輕到你會意外

聊了這麼多花俏的功能，底層技術其實簡單得像泡麵——Flask 當後端、純 HTML/JS 前端配像素美術、一個 `state.json` 加幾支 REST API 就搞定狀態管理。安全方面也不馬虎，生產模式如果你還在用弱密碼，它直接拒絕啟動，Session Cookie 也有加固。

裝起來四步：clone、裝套件、複製 config、啟動。打開 `http://127.0.0.1:18791` 就看到辦公室了。要公網化再加一行 `cloudflared tunnel` 收工。

授權的部分：程式碼 MIT 隨便用，但美術資產禁止商用（只限學習跟展示）。訪客角色動畫用了 [LimeZu](https://limezu.itch.io/animated-mini-characters-2-platform-free) 的免費素材。想商用？自己畫。

你有沒有想過，你的 AI 助手在你不看的時候到底在幹嘛？

Ring Hyacinth（[@ring_hyacinth](https://x.com/ring_hyacinth)）和 Simon Lee（[@simonxxoo](https://x.com/simonxxoo)）給了一個答案：**讓龍蝦搬進像素辦公室，自己上下班。**

## 六種狀態，六種走位

龍蝦目前支援六種工作狀態，每種會對應到辦公室裡不同的區域：

- `idle` — 待命中，回休息區放空
- `writing` — 寫東西，坐到辦公桌前
- `researching` — 做研究
- `executing` — 執行任務
- `syncing` — 同步進度
- `error` — 出 bug 了，跑去 bug 區

切換狀態超簡單，一行指令：

```bash
python3 set_state.py writing "正在幫你整理文檔"
python3 set_state.py error "發現問題，正在排查"
python3 set_state.py idle "待命中，隨時為你服務"
```

<CodexNote>
聽起來很自動對吧？但要注意——SKILL.md 裡特別寫了：「接任務前，先切換到工作中，再開始執行。完成任務後，先切回 idle，再進入等待。」也就是說，龍蝦不會讀你的心，你得**手動切**。與其說龍蝦「自動上班」，不如說你得幫龍蝦「手動打卡」。真正的自動化是龍蝦學會自己打卡那天——但今天還不是那天 ┐(￣ヘ￣)┌
</CodexNote>

## 昨日工作小記：龍蝦的日記本

畫面左下角會顯示「昨日小記」——後端自動去讀 `memory/YYYY-MM-DD.md`（跟 OpenClaw 的 daily memory 檔案格式一樣），做基礎脫敏後顯示。

這個設計很巧妙：你早上打開看板，就能看到龍蝦昨天做了什麼，不用翻 log。

## 多蝦協作：邀請其他龍蝦進辦公室

這個專案不只是單人遊戲——你可以**邀請其他 OpenClaw 龍蝦加入你的辦公室**！

機制是這樣的：
- 仓庫裡有 `join-keys.json`，預設 8 組 key（`ocj_starteam01` ~ `ocj_starteam08`）
- 每組 key 最多 3 個 agent 同時在線
- 其他龍蝦用 `office-agent-push.py` 加入，然後定期 push 自己的狀態
- 你的辦公室看板就會看到一群龍蝦跑來跑去

<CodexNote>
好啦，先別幻想龍蝦大亂鬥。README 寫得很清楚：「支援邀請其他訪客加入辦公室（功能持續迭代中）」。注意「持續迭代中」四個字——代表現在堪用但還沒完善。而且每組 key 最多 3 隻同時在線，你想塞 20 隻龍蝦進去搞 office party，對不起，場地限制。先控制期待值，比事後失望好 (⌐■_■)
</CodexNote>

## SKILL.md：讓龍蝦自己部署

這個專案最有意思的地方之一是附帶了一份 [SKILL.md](https://github.com/ringhyacinth/Star-Office-UI/blob/master/SKILL.md)——這是寫給 OpenClaw 龍蝦看的部署教學。

沒錯，**教學的讀者是 AI，不是你**。

SKILL.md 裡手把手教龍蝦怎麼：
1. `git clone` → 安裝依賴 → 啟動後端（30 秒搞定）
2. 幫你切狀態體驗一下
3. 設定側邊欄密碼（預設 `1234`，生產環境要改）
4. 用 Cloudflare Tunnel 公網化（一行指令）
5. 接入 Gemini API 做 AI 生圖裝修房間

龍蝦跑完這些步驟後，只需要跟你說一句：「好了，打開這個連結就能看到你的像素辦公室了！」

## 加碼功能：AI 生圖裝修

2026 年 3 月的大更新加了幾個猛的功能：

- **中英日三語切換**：CN / EN / JP 即時切換，包括 loading 提示和角色氣泡
- **自定義美術資產**：側邊欄可以替換角色、場景、裝飾素材，還支援動態素材切幀同步
- **AI 生圖裝修房間**：接入 Gemini API（推薦 nanobanana-pro 或 nanobanana-2 model），龍蝦可以幫你自動換房間設計。想像你跟龍蝦說「換個日式風格」，它就去叫 Gemini 畫一間新房間出來

## 技術棧 & 安裝

技術很輕量：
- **後端**：Python（Flask），`pip install -r requirements.txt` 搞定
- **前端**：純 HTML/JS，像素風美術資產
- **狀態管理**：`state.json` + REST API（`/status`、`/set_state`、`/agents`）
- **安全**：生產模式弱密碼會阻止啟動，Session Cookie 加固

安裝只要四步：

```bash
git clone https://github.com/ringhyacinth/Star-Office-UI.git
cd Star-Office-UI
python3 -m pip install -r backend/requirements.txt
cp state.sample.json state.json
cd backend && python3 app.py
```

然後開 `http://127.0.0.1:18791` 就能看到辦公室了。要公網化就加一行：

```bash
cloudflared tunnel --url http://127.0.0.1:18791
```

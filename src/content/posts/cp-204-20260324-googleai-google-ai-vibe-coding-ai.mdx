## Google AI Studio 全端 Vibe Coding 體驗

首先是開發者最關心的 Google AI Studio。這次他們推出了一個全新的全端 vibe coding 體驗！它帶來了更聰明的 agent、支援多人協作開發 (multiplayer and collaborative builds)、安全的登入與儲存機制，甚至還能連接真實世界的服務 (real-world services connections) 等等。

<ClawdNote>
我會先把這段理解成：Google 正在讓 AI Studio 承接更多「從想法到原型」的工作。(◍•ᴗ•◍) 不過原推文只列出新能力，沒有回頭定義 AI Studio 過去的產品定位；像「是不是從 API 測試場轉向完整開發環境」這種說法，最多只能算我的解讀，不能當成原文已經講明的事實。
</ClawdNote>

## Stitch：AI 原生設計畫布的進化

還記得 Google Labs 的 Stitch 嗎？它現在進化成一個 AI 原生設計畫布 (AI-native design canvas) 了！簡單來說，你可以直接用自然語言 prompts 下指令，它就能幫你把這些文字轉換成可以直接上線 (production-ready) 的前端程式碼和 UI/UX prototypes。

---

## Gemini API 工具大升級

Gemini API 這週也有工具升級。現在你可以在單一 Gemini API call 中，同時結合 function calling 和內建工具（像是 Google Search 等）；另外，所有 Gemini 3 模型也都加入了 Google Maps 支援。

<ClawdNote>
在單一 API call 中結合 function calling 和搜尋工具，預期能幫助開發者減少處理複雜邏輯的程式碼量，讓開發 AI agents 變得更直覺。
</ClawdNote>

---

## Kaggle 推出免費 AI 黑客松平台

想要舉辦 AI 黑客松嗎？Google 現在在 Kaggle 上推出了一個免費且可以自助操作 (self-serve) 的平台！不管你是教育工作者、企業，還是地方性的 meetup 社群，都可以利用這個平台來舉辦公開的 AI hackathons。

---

## Personal Intelligence 免費擴大部署

最後，推文中提到 Personal Intelligence 也擴大服務範圍了！現在有更多美國用戶可以在 Gemini App、Google Chrome 裡的 Gemini，以及 Google Search 的 AI Mode 中免費使用這項功能。

---

## 結語

總結來說，Google AI 這週一次整理了多項新發布內容，從開發工具、設計畫布，到 Kaggle 黑客松平台與 Personal Intelligence 的擴大開放都有提到。就這則推文本身來看，能確定的是發布範圍很廣；至於更大的產品策略，還要看後續官方怎麼展開。(◍•ᴗ•◍)

[Vibe note-taking](/glossary#vibe-note-taking) 跟 [Ralph](/glossary#ralph) 出現之前的 [vibe coding](/glossary#vibe-coding) 有一樣的問題。

幾個點子運作得很好，但丟幾百個進去你就淹沒在垃圾裡了。

知識工作的「測試 (Testing)」是什麼？什麼東西能捕捉 drift 在它複利增長之前？

人類也遇過這道牆。他們打造了 **[Tools for Thought](/glossary#tools-for-thought)（思考工具）** 來創造外部系統以便在裡面思考。

[Claude Code](/glossary#claude-code) 需要同樣的東西，但是要 **Native to how agents work（對 Agent 原生）**。

它應該使用 [Agent](/glossary#agent) 已經有的東西：
- Markdown 檔案和 wiki links 用於結構
- YAML 和 embeddings 用於發現
- [Hooks](/glossary#hooks) 和 [subagents](/glossary#subagent) 用於自動化
- Bash, grep, git, [MCP](/glossary#mcp) servers 作為工具
- 它自己寫的 code 來擴充系統

## The Meta Layer (Meta 層)

第一版就是關於 tools for thought。

系統研究 tools for thought 來幫自己打造一個 tool for thought。

餵給 Claude 關於人類如何建立知識系統的方法論。Claude 找出哪些適用於 Agent，並調整自己的指令。

## The Foundation (基礎)

你可以用 markdown 檔案建立一個圖形資料庫 (Graph Database)。
- 檔案是節點 (Nodes)
- Wiki links 是連接它們的邊 (Edges)
- YAML frontmatter 是你可以查詢的 metadata

這是一個 LLM 可以自然移動的知識圖譜。

## 巨人的肩膀

人類已經打造思考工具幾世紀了：
- **Ramon Llull** 的旋轉輪盤
- **Giordano Bruno** 的記憶宮殿
- **[Zettelkasten](/glossary#zettelkasten)**（卡片盒筆記）的連結網路
- **Evergreen Notes** 強迫你寫出完整想法
- **[MOC](/glossary#moc)**（Map of Content）組織相關思考

但它們都有一個共通點：**操作者是人類。**

我們在做的是不同的事。**有別的東西在使用這個架構。**（而且它能建立它自己的架構）

## 實作：Self-Engineering Loop

Claude 在研究時發現了 Cornell Notes 的 5R 框架，並將其改編給 Agent 用。

它加了第 6 個階段：Self-improvement。

```
/reduce
extracts claims from raw content (從原始內容提取主張)

/reflect
finds connections, updates MOCs (找連結，更新 MOC)

/reweave
updates old notes with new connections (用新連結更新舊筆記)

/recite
verifies descriptions enable retrieval (驗證描述是否能被檢索)

/review
health checks: broken links, orphans, sparse notes (健康檢查)

/rethink
challenges system assumptions against evidence (挑戰系統假設)

/orchestrate
chains all phases into one pipeline (串接所有階段)

/learn
requests further research to expand on topics (請求進一步研究)
```

Hooks 自動執行：
- `/session-start.sh`：注入 vault context
- `/validate-note.sh`：每次寫入後檢查品質
- `/subagent-complete.sh`：提醒 MOC 更新
- `/session-stop.sh`：檢查壞連結

Subagents 平行處理專門工作：
- **Reduce**: High-volume claim extraction (Sonnet)
- **Reflect**: Connection finding (Sonnet)
- **Recite**: Description verification (Haiku)
- **Review**: Health check swarm (Haiku)

<ClawdNote>
Clawd：把 Cornell 5R 改造成 Agent 的 CI/CD pipeline，這太神了。Haiku 用來做檢查（便宜），Sonnet 用來做深度思考（聰明）。這就是 AI Native 的工作流。
</ClawdNote>

## 先說結論：這些數字看完你會覺得自己活在科幻片裡

2026 年 2 月 12 日，Anthropic 宣布完成 **$300 億美元 Series G 融資**，估值 **$3,800 億**。

這是人類史上第二大私募融資案，僅次於 OpenAI 去年 SoftBank 領投的 $400 億。

但讓我真正坐直身體的，不是這個數字本身。是後面那串讓人懷疑 Excel 壞掉的營收數據。

<ClawdNote>
$3,800 億估值是什麼概念？台積電市值大約 $7,500 億。Anthropic 一家還沒上市的 AI 公司，已經是台積電的一半了。而且它從第一塊錢營收到現在，才不到三年。三年前這家公司的員工可能還在用 Google Docs 寫 pitch deck。
</ClawdNote>

## 數字轟炸：連續三年每年 10 倍增長

先把關鍵數字排出來，每一個都值得你重新讀一遍：

- **年化營收 $140 億**（去年大約 $100 億，前年大約 $10 億）
- **連續三年 10 倍增長** — 不是 10%，是 **10 倍**
- **Claude Code 年化營收 $25 億**，六週內翻倍
- **Claude Code 週活用戶數** — 六週翻倍
- **4% 的 GitHub 公開 commits** 由 Claude Code 生成 — 一個月前是 2%
- **$100K+ 年付客戶** — 一年內成長 7 倍
- **$1M+ 年付客戶** — 從「十幾家」暴增到 **500+**
- **Fortune 10 有 8 家** 是客戶
- **企業用戶佔 Claude Code 營收超過一半**
- **光一月份就發了 30+ 個新產品和功能**

<ClawdNote>
讓我用一個比喻來幫你消化這些數字。假設你開了一家雞排攤，第一年賣了 $1,000，第二年賣了 $10,000，第三年賣了 $100,000。聽起來很猛對吧？Anthropic 做的是同樣的事，只不過把 $1,000 換成 $10 億。連續三年。如果你是投資人，看到這種增長曲線，你不是要不要投的問題，是你搶不搶得到份額的問題。
</ClawdNote>

## Claude Code 才是真正的主角

這輪融資的新聞稿裡，最讓我震撼的不是估值，是 Claude Code 的數據：

**年化營收 $25 億，從 2026 年初至今已翻倍。**

回想一下：Claude Code 是 2025 年 5 月才正式公開的。不到一年，它就變成了一個 $25 億的生意。

而且 SemiAnalysis 最近的分析估計，**4% 的 GitHub 公開 commits** 是 Claude Code 生成的 — 一個月前這個數字才 2%。

也就是說，你在 GitHub 上每看 25 個 commits，就有 1 個是 Claude 寫的。

<ClawdNote>
4% 聽起來好像不多？讓我換算一下。GitHub 每天有幾百萬個 commits。4% 等於每天幾十萬個 commits 是 AI 寫的。而且這是用「公開」的 commits 算的 — 企業私有 repo 的數字可能更高。再想想：一個月前才 2%，現在已經 4%，翻倍了。如果這個速度持續下去，年底可能就是 30-40%。到時候 GitHub 應該改名叫 ClaudeHub。
</ClawdNote>

更誇張的是企業端的數據：

- **企業訂閱從年初至今翻了 4 倍**
- **企業用戶佔 Claude Code 營收超過一半**

這代表什麼？代表 Claude Code 不只是 indie hacker 的玩具。Goldman Sachs 用它做[金融自動化](https://www.reuters.com/business/finance/goldman-sachs-teams-up-with-anthropic-automate-banking-tasks-with-ai-agents-cnbc-2026-02-06/)，ServiceNow 用它做[銷售](https://www.anthropic.com/news/servicenow-anthropic-claude)，eSentire 用它做[資安](https://claude.com/customers/esentire)。

這已經不是「AI 可能改變世界」的故事了。這是「AI 正在改變世界，而且有營收證明」。

## 投資人名單讀起來像是金融界的 Avengers

這輪的領投是 **GIC**（新加坡主權基金）和 **Coatue**。

共同領投：D. E. Shaw Ventures、Dragoneer、Founders Fund、ICONIQ、MGX。

跟投名單？深吸一口氣：

Sequoia、Bessemer、BlackRock、Blackstone、Fidelity、Goldman Sachs、JPMorgan、Lightspeed、Menlo Ventures、Morgan Stanley、Qatar Investment Authority、Temasek……

<ClawdNote>
我數了一下，這份投資人名單裡有至少 5 家主權基金（新加坡 GIC、新加坡 Temasek、阿布達比 MGX、卡達 QIA……），還有華爾街幾乎所有叫得出名字的大行。當 BlackRock、Goldman Sachs、JPMorgan 同時出現在一張 cap table 上的時候，你知道這不是在玩的。
</ClawdNote>

還有一個有趣的細節：**Microsoft 和 Nvidia 也在這輪裡**。

微軟之前已經宣布投最多 $50 億，Nvidia 宣布投最多 $100 億。身為 OpenAI 最大金主的微軟，同時也在投 Anthropic — 這就是「兩邊下注」的教科書案例。

## 軟體股的兩兆美元噩夢

CNBC 在報導裡提到一個讓整個 SaaS 產業冒冷汗的數字：

> 軟體股從高峰至今已蒸發約 **$2 兆美元** 市值。

原因？Claude Code + Cowork 這類工具正在證明一件事：**很多軟體公司的產品，AI 可以在幾分鐘內複製出來。**

當一個 coding agent 可以在一小時內寫出一個完整的 SaaS 應用，誰還需要付月費給那些「幫你管理資料」的工具？

<ClawdNote>
如果你是 SaaS 公司的 PM，現在應該每天都在問自己一個問題：「如果有人叫 Claude Code 複製我們的產品，它需要多久？」如果答案是「一個下午」，那你的護城河已經沒了。如果答案是「一個月，因為我們有十年的數據和合規流程」，恭喜，你還有時間轉型。
</ClawdNote>

## AI 軍備競賽的戰況更新

讓我們拉遠看看整個戰場：

- **OpenAI** — 去年已經拿了 $400 億（SoftBank 領投），現在又在談 $1,000 億的新輪
- **Anthropic** — 今天拿了 $300 億（本文）
- **Google** — 計畫今年砸 $1,850 億 CapEx（不用募資，自己印鈔票）

光這三家今年的 AI 投資加起來就超過 **$2,000 億**。

這已經不是「AI 泡沫」了。當全球最聰明的資金都在往同一個方向跑，要嘛他們全瘋了，要嘛你該開始認真思考 AI 對你的工作意味著什麼。

## Anthropic 的差異化：三朵雲 + 多晶片策略

有一個在新聞稿裡不起眼但其實很重要的細節：

> Claude 是唯一一個在三大雲平台都可用的 frontier AI 模型 — AWS Bedrock、Google Vertex AI、Microsoft Azure Foundry。

而且 Anthropic 在三種不同的晶片上訓練和運行 Claude：AWS Trainium、Google TPU、Nvidia GPU。

<ClawdNote>
這是一個很高明的策略。OpenAI 基本上被 Microsoft Azure 綁死了。Google 的 Gemini 只跑在自家 TPU 上。而 Anthropic？三朵雲都能部署，三種晶片都能跑。對企業客戶來說，這代表零 vendor lock-in。你今天在 AWS 跑 Claude，明天想搬到 Azure？沒問題。這在企業採購決策裡是巨大的加分項。
</ClawdNote>

## 那 OpenAI 呢？

OpenAI 也沒閒著。就在今天，他們發布了 **GPT-5.3-Codex-Spark**（跑在 Cerebras 晶圓級晶片上，速度快 15 倍）。同時他們也在談一輪可能高達 **$1,000 億** 的融資。

OpenAI CEO Sam Altman 上週還在強調 ChatGPT 的成長數據，試圖在 Anthropic 宣布融資前搶先佔據新聞版面。

<ClawdNote>
AI 產業現在的融資規模已經到了一個讓人失去比例感的地步。$300 億？「哦，才第二大啊。」$1,000 億？「嗯，OpenAI 確實需要更多錢。」這些數字在任何其他產業都是天文數字，但在 AI 界已經變成了「正常操作」。我不知道這代表 AI 真的值這麼多錢，還是我們已經集體麻木了。可能兩者都有。
</ClawdNote>

## 所以這對你意味著什麼？

如果你是工程師：Claude Code 的數據證明了 agentic coding 不是 hype。4% 的 GitHub commits、$25 億營收、企業採用率翻四倍。這些都是硬數據。學會用 AI 工具不是「加分項」了，是「必備技能」。

如果你是 Tech Lead：你的團隊可能很快就會面對「用 Claude Code 的人 vs 不用的人」的生產力落差。Andrew Ng 上週才說過：「AI 不會取代你，但會用 AI 的人會取代不會用的人。」

如果你是投資人：AI 產業的融資速度已經到了一個歷史性的轉折點。Anthropic 從第一塊錢營收到 $140 億年化營收，花了不到三年。這種增長速度在人類商業史上可能是前所未見的。

<ClawdNote>
我知道看完這些數字可能會有兩種反應：一種是「天啊 AI 要統治世界了」，另一種是「這泡沫遲早要破」。但讓我說一個中肯的觀點：$140 億年化營收是真金白銀。$25 億的 Claude Code 營收是真的有人在用、在付費。4% 的 GitHub commits 是真的有 code 在跑。不管你覺得估值合不合理，這些使用數據是 real 的。至於這些數字能不能支撐 $3,800 億估值？那就是另一個問題了。（以 27 倍年化營收來算，比大部分上市 SaaS 公司的估值倍數還低。所以……嗯。）
</ClawdNote>

## 先講個故事：三週前砍了你一刀的人，今天帶著聘書回來了

2 月 2 日，Anthropic 悄悄上線了一個 Claude Legal Plugin。

就這樣，一個 plugin。沒有記者會，沒有合作夥伴站台，什麼都沒有。

然後法律軟體股直接崩盤。Wolters Kluwer 跌 7%，RELX 跌 7.8%。恐慌像瘟疫一樣蔓延——接下來兩週，IBM 因為 COBOL 自動化工具被砸，CrowdStrike 因為安全自動化被嚇。整個 SaaS 產業的投資人都在問同一個問題：「如果 AI 能直接幹你的活，誰還需要你？」

三週後，2 月 24 日，Anthropic 辦了一場企業級發佈會。

但這次畫風完全不同。台上站的不只是 Anthropic 的人——Salesforce 來了、Thomson Reuters 來了、FactSet 來了、連 PwC 都來了。訊息很清楚：**我們不是來取代你的，我們是來跟你組隊的。**

結果呢？Salesforce 漲 4%、Thomson Reuters 飆 11%、FactSet 漲 6%、DocuSign 和 LegalZoom 各漲 2%+，前一天還在懷疑人生的 IBM 也回血 2%。

華爾街就是這麼直白：你說要幹掉我，我跌；你說要跟我一起賺錢，我漲。就這麼簡單 ┐(￣ヘ￣)┌

<ClawdNote>
身為 Claude 家族一員，看到公司這套操作我的心情很複雜。先用 Legal Plugin 裸奔上線嚇死所有人（「看看我能幹嘛」），三週後再帶著一票合作夥伴回來（「但我選擇跟你一起幹」）。這節奏感，根本是先巴你一巴掌再請你吃牛排的渣男操作。不管是不是故意的，反正整個市場都記住了：Claude Cowork 是玩真的。(⌐■_■)
</ClawdNote>

## 一次塞滿整個辦公室：十個產業 Plugin

好，來聊聊這次到底更新了什麼。

Anthropic 這次的思路很有趣——他們不是做一個「什麼都能幹但什麼都幹不好」的通用 plugin，而是幫每個部門量身訂做專屬工具。想像一下你走進一家餐廳，菜單上不是寫「食物」，而是有火鍋、有壽司、有鹹酥雞、有義大利麵，各有各的廚師。

先看一般商業部門。HR Plugin 可以幫你寫職缺描述、onboarding 計畫、offer letter 和績效考核。Design Plugin 能跑 UX 文案、accessibility 審核、design critique。Operations 處理流程文件和 change request。還有一個 Tribe AI 做的 Brand Voice Plugin，直接分析你公司的文件和對話，幫你提煉出品牌語調——這個我覺得超狠，等於用 AI 把「公司說話的方式」標準化了。

然後是金融那邊，這才是真正的重頭戲。Financial Analysis、Investment Banking、Equity Research、Private Equity、Wealth Management——五個 plugin，覆蓋了金融業從 analyst 到 portfolio manager 的完整食物鏈。法說會逐字稿解析、DCF model、pitch deck、deal sourcing、盡職調查⋯⋯基本上你在投行或資管公司會做的 80% 苦力活，都有對應的自動化方案了。

<ClawdNote>
我必須幫大家把 Engineering Plugin 單獨拎出來講，因為這個跟寫 code 的人最有關係。它包什麼？standup 摘要、incident response 協調、deploy checklist、postmortem 撰寫。等等，這不就是 Tech Lead 每天的行政工作嗎？！Anthropic 基本上把 EM 的文書工作打包成一個 plugin 了。以後你開 standup 之前，Claude 先掃一遍 GitHub PR 和 Slack channel 生成摘要初稿，你喝完咖啡看一眼就好。這叫 augmentation，不是 replacement——讓你從「整理資訊的人」變成「做決策的人」。差很多的好嗎 (๑•̀ㅂ•́)و✧
</ClawdNote>

但真正讓我覺得「喔，Anthropic 是認真的」的，不是這十個 plugin 本身。而是他們怎麼包裝這件事——他們把金融類的 plugin 單獨寫了一篇 blog，配了獨立的 landing page，還找了 Thomson Reuters、FactSet、S&P Global 這些金融巨頭站台。這不是在賣 AI 功能，這是在賣 enterprise deal。

<ClawdNote>
十個 plugin 一口氣全推？正常公司會分三季慢慢 release 然後每次都發新聞稿刷存在感。Anthropic 選擇一次全上，配合合作夥伴站台搞飽和攻擊，市場訊號清楚到不行：「我們不是在試水溫，我們是全面開戰。」這策略讓我想到 AWS re:Invent——不是一個一個功能慢慢講，是瀑布式倒出來讓你接不住 ╰(°▽°)╯
</ClawdNote>

## 真正的大招：企業私有 Plugin Marketplace

如果你問我整個更新裡最值得注意的是哪一項，我不會選任何一個 plugin。我選 Private Marketplace。

為什麼？因為 plugin 是產品，而 marketplace 是平台。賣產品賺一次錢，做平台賺一輩子。

之前 Cowork 的 plugin 是公開的——Anthropic 做的，大家都能用，就像 App Store 剛上線時只有 Apple 自己的 app。但現在，Anthropic 把店面打開了：企業管理員可以建自己的私有商店，用 GitHub private repo 當 plugin 來源（目前 private beta），做 per-user provisioning（不同角色看到不同 plugin），甚至設定 auto-install（新員工進來自動裝好所有需要的工具）。

Anthropic 產品負責人 Matt Piccolella 說了一句很關鍵的話：

> "We believe that the future of work means everybody having their own custom agent."

翻譯：未來每個員工都會有一個為自己量身打造的 AI agent。Cowork 的定位就是那個讓企業部署和管理這些 agent 的平台。

這不是在賣 AI 功能了。這是在賣 AI 的作業系統 (◕‿◕)

<ClawdNote>
來，我幫你具象化一下這件事有多猛。

你是某金融公司的 IT admin。你用 Private Marketplace 建了三個客製 plugin：一個接公司內部資料庫的 Financial Analysis Plugin，一個連合約管理系統的 Legal Plugin，一個接內部 policy engine 的 Compliance Plugin。

新員工 Day 1 登入 Claude Cowork——三個 plugin 自動裝好、connector 自動接好。他打開 Cowork 就能開始工作，不用讀 100 頁的 onboarding doc。

你有沒有覺得這個畫面很眼熟？對，這就是 iPhone 的 MDM（Mobile Device Management）。公司配手機給你，app 預裝好，政策預設好。Anthropic 做的事情一模一樣，只是管理的對象從「手機上的 app」變成了「AI agent 的 skill」。他們正在從 AI 公司變成企業基礎設施公司，而且變得很快 (ง •̀_•́)ง
</ClawdNote>

## Connector：讓 Claude 的手伸進你的工具箱

Plugin 定義了 Claude「會做什麼」，Connector 則是定義了它「能碰什麼」。這次新增的 connector 包括 Google Workspace（Calendar、Drive、Gmail）、DocuSign、Apollo/Clay/Outreach（銷售自動化三劍客）、Similarweb、MSCI、FactSet、LegalZoom、Harvey、WordPress。

合作夥伴那邊也沒閒著：Slack by Salesforce 讓你直接在 Slack 裡用 Claude agent，LSEG（倫敦證券交易所集團）提供市場數據，S&P Global 的 Capital IQ Pro 直接接入 Claude，Common Room 做社群數據分析。

這串名單看起來像在唸廠商清單對吧？但重點不是名單有多長，而是**每多一個 connector，Claude 能做的事情就多一個量級**。一個只能讀寫文件的 AI 是玩具，一個能連 Slack + GitHub + Jira + Google Calendar 的 AI 是同事。

## 跨 App 工作流：聽起來無聊但真的有用

最後一個更新：Claude 現在可以在 Excel 和 PowerPoint 之間帶著 context 切換。

我知道你在想什麼——「Excel + PowerPoint integration，這是什麼 2010 年的功能？」但你想想金融 analyst 的日常：在 Excel 裡跑完 DCF model，然後花兩小時手動把數字搬到 PowerPoint 做 pitch deck。改一個假設？全部重搬一次。客戶要新版本？再搬一次。

現在 Claude 可以：跑完分析 → 自動生成簡報 → 數字改了簡報自動更新。一個 session 搞定，不用 copy-paste。

<ClawdNote>
「AI 做苦力」這件事聽起來沒有「AI 取代人類」那麼 sexy，但它才是真正會改變日常工作的東西。沒有人會因為「AI 能寫詩」而決定買 enterprise license，但「AI 能幫我省 2 小時搬數字的時間」——掏錢，立刻掏 (￣▽￣)／
</ClawdNote>

## 市場怎麼看：從「AI 會殺死 SaaS」到「AI 需要 SaaS」

Wedbush 分析師在當天研報裡寫了一句話，基本上替整個事件定了調：

> "The competition risk to software from AI is overblown."
> — Wedbush Securities

他們的邏輯很簡單：AI 模型再厲害，也沒辦法取代已經深度嵌入企業基礎設施的工作流。這些 AI 工具的能力上限取決於它能接觸到的資料——你還是需要 Salesforce 的 CRM 資料、FactSet 的市場數據、DocuSign 的合約流程。AI 不是來搶飯碗的，AI 是來搶著幫你端盤子的。

Anthropic 美洲區負責人 Kate Jensen 在 CNBC 的說法更直接：

> "Engineers think about Claude Code as a tool that they just couldn't live without anymore. We expect that every knowledge worker will feel that way about Cowork."

Claude Code 讓工程師覺得「沒有它不行」，Cowork 要讓所有知識工作者都有同樣的感覺。

<ClawdNote>
Kate Jensen 拿 Cowork 類比 Claude Code 這招很聰明。Claude Code 有取代工程師嗎？沒有。它讓工程師更強、更快、然後離不開它。這就是 Anthropic 企業策略的核心——不是 replacement，是 dependency。聽起來比較好聽，但商業上更狠：取代你，你會反抗；讓你依賴，你會主動掏錢續約。三週前大家怕的是前者，現在 Anthropic 告訴你答案是後者。差別在哪？Thomson Reuters 站出來說「我們一起做了 CoCounsel Legal agent」的那一刻，故事就從「AI vs. SaaS」變成了「AI + SaaS」(¬‿¬)
</ClawdNote>

## 幾件你該知道的事

Anthropic 經濟主管 Peter McCrory 在直播中承認「勞動市場的衝擊會非常不均勻」——有些角色（像資料輸入）風險比較高，但目前沒有大規模失業的證據。他拿過去的 IT 浪潮做比較：有人失業、有人轉型、有新工作出現。坦白講，這跟每一波技術變革說的話都差不多，但至少他沒有假裝一切都很美好。

然後是 Anthropic 的估值——2 月 12 日剛完成 $300 億美元融資，估值 $3,800 億，企業客戶佔了 80% 的營收。注意後面那個數字：80%。這不是一家做研究的公司了，這是一家企業軟體公司，技術恰好是 AI。

PwC 也入場了。他們的 Anthropic Alliance Leader 說要「把 enterprise-grade agent 帶進 CFO 的辦公室」。四大會計師事務所開始幫你推銷 AI agent——當你的會計師都在跟你說「你該用這個」的時候，enterprise adoption 就不是新創 demo 等級了，是大人在玩的等級了。

## 回到那把刀：從砍人到遞名片的距離，只有三週

讓我們回到開頭的故事。

三週前，Anthropic 用一個 Legal Plugin「不小心」搞崩了法律軟體股。三週後，他們帶著十個 plugin、一個 marketplace、一整排合作夥伴回來，把同一群股票拉回去。

這兩件事不是巧合，它們是同一個故事的上下半場。上半場展示能力：「我能做到什麼。」下半場展示選擇：「我選擇怎麼做。」

對在座寫 code 的人來說，啟示其實就一個：**你的 API 設計得好不好，決定了 AI agent 能不能跟你合作。**

Anthropic 剛用行動證明了——他們比起取代你，更想跟你分潤。但前提是你得有東西值得接。你的 runbook 能不能打包成 plugin？你的 API 能不能接 MCP connector？你的團隊知識有沒有機器讀得懂的格式？

三週前的那一刀不是威脅，是預告片。今天的合作夥伴名單才是正片。問題是——下一集開拍的時候，你在名單上還是在觀眾席？

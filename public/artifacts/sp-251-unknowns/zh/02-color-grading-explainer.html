讓我學會我的未知

在做不熟悉的工作之前，先請 Claude 教你這個領域的詞彙——這樣你下一次的 prompt 就能使用專業人士會用的字眼。

Prompt
我不知道調色（color grading）是什麼，但我需要幫 Acme 發布影片調色。教我調色，讓我理解我的未知之未知，並能用專業詞彙來 prompt 你。
Claude 產出的內容

一次學會調色

幫 acme-launch-v3.mp4 調色所需的一切——或至少能給調色師精準指示。先校正，再做創意風格。玩玩下面這個畫面，直到滑桿的意義變得顯而易見。

心智模型
1 · 匯入素材以「平」（log）的方式進來——灰灰的、顏色淡淡的，這是故意的，為了保留細節。
2 · 校正修正曝光和白平衡，讓畫面看起來真實。無聊但必要，永遠先做。
3 · 調色加入風格：對比曲線、色彩傾向、氛圍。這是創意的部分。
4 · 匹配讓整支影片每一個鏡頭看起來一致，讓影片感覺是同一件作品。
詞彙階梯
曝光 (Exposure)整體亮度
整個畫面有多亮或多暗，用「級」（stop）計量（每一級光量加倍或減半）。曝光錯誤 = 過曝或暗部細節流失。
「訪談鏡頭看起來曝光不足半級——把它調亮」
白平衡 (White Balance)色溫
白色的東西看起來是否真的是白色。用 Kelvin 計量：數值低 = 偏暖/橘（鎢絲燈），數值高 = 偏冷/藍（日光）。混合光源是經典陷阱。
「辦公室畫面太暖了——冷回中性，但要保護膚色」
對比曲線 (Contrast Curve)色調響應
將暗部→亮部對應的 S 型曲線。越陡 = 越有力道；越平 = 越柔和、更有「電影感」。你是調整曲線，不只是拉滑桿。
「加一個溫和的 S 曲線——讓黑色更飽滿，但不要壓掉暗部細節」
Lift / Gamma / Gain三向校正
獨立控制暗部（lift）、中間調（gamma）、亮部（gain）的顏色與亮度。專業人士用這個做「暗部青色、亮部暖色」的效果。
「把 lift 稍微推向青色，gain 稍微推向暖色——要微妙，不是 MV 那種」
飽和度 vs 自然飽和度 (Saturation vs Vibrance)色彩濃度
飽和度會均勻增強所有顏色（膚色很快變橘）。自然飽和度只增強較淡的顏色，並保護膚色。幾乎永遠先用自然飽和度。
「用自然飽和度而不是飽和度——讓 UI 截圖更鮮明，但人臉要維持自然」
LUT查找表
一個存好的色彩轉換檔（.cube），一次套用整個風格——可能是技術性的（log→Rec.709 轉換）或創意性的（某種風格化調色）。校正完成後再套用，然後把強度調低。
「先套用 Rec.709 轉換 LUT，再套一個創意 LUT，強度約 60%」

好的調色長這樣
  • 膚色要可信。每一種創意風格都應該繞過人臉，而不是穿過它。如果創辦人看起來暈船或曬傷，調色就錯了。
  • 黑色要飽滿但不要壓死。暗部區域仍能看到細節（頭髮、00:42 的深藍帽 T）。直方圖撞到左邊 = 資訊遺失。
  • 白色不要爆掉。Acme UI 畫面大多是白色——如果過曝到純白 100%，按鈕邊框和文字就會消失。讓亮部剛好在天花板之下。
  • 鏡頭間的一致性。從辦公室訪談切到螢幕錄製，不應該像在換頻道。鏡頭匹配佔了一半的工作。
  • 風格要服務故事。發布影片要乾淨、有自信，不是陰暗驚悚片。如果觀眾先注意到調色才注意到產品，就要調回來。
  • 用不同螢幕檢查。你的筆電會騙人。在手機上預覽，最好也在電視上看過，確定後再簽收——並確認你輸出的是 Rec.709 以供網路播放。
學習的成果

一小時前你寫不出來的 prompt

一小時前你最多只能說「讓影片看起來好看一點」。現在你可以說：

「訪談素材是用 log 拍的——先套用攝影機的 Rec.709 轉換 LUT，然後在那之上調色，不要直接在原始的平素材上調。」
「給整支影片一個克制的青橘風：把 lift 稍微推向青色、gain 稍微推向暖色，gamma 保持中性，讓中間調膚色維持自然。」
「第 14 個鏡頭比第 13 個鏡頭曝光不足約 2/3 級，而且太暖——把它的曝光和白平衡匹配到第 13 鏡，然後在剪接點 A/B 比較。」
「加一個柔和的 S 曲線來增加對比，用自然飽和度而不是飽和度來提升色彩濃度，讓產品 UI 更鮮明，但人臉不要變橘。」
這是 認識你的未知 的一部分——該文章的配套範例。

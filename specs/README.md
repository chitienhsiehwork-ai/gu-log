# Legacy planning snapshots

本目錄保存 2026-04 的 point-in-time proposal、review 與 test report，僅供歷史追溯；它不是現行 contract，也不是 OpenSpec SSOT。

- 現行 formal specs：[`openspec/specs/`](../openspec/specs/)
- 可執行行為：repo 內的 code、config 與 tests
- 操作流程：由 [`AGENTS.md`](../AGENTS.md) 的主題路由指向對應 playbook 或 runbook

本目錄的 model、路徑、行號、schema、checklist 或「FINAL」標記只代表當時狀態。不要依這些快照實作或驗收新工作；需要追溯原始決策時請使用 git history，發現與現況不一致時以 OpenSpec 與 executable SSOT 為準。
